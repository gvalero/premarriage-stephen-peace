# Stephen & Peace Roles in Marriage Homework Page

This folder is a GitHub Pages-ready static site for Session 4 homework.

Three jobs the homework does:
1. Closes the loop on the **repair covenant** we didn't get to in Session 3.
2. Looks at **how the wedding/honeymoon decision was made** as a live case study.
3. Sets up the next chapter: **Roles in Marriage** (book pp. 36-53).

Important privacy note: this page does not send responses anywhere by itself. GitHub Pages is static and has no private form backend. The form stores answers in the browser only, then lets Stephen and Peace copy or download their responses and send them back manually.

Suggested publish path:

1. Create a private or low-visibility GitHub repository under your personal account (or push to the same repo used for Session 3 under a `/session-04/` subfolder).
2. Copy the contents of this `github-page` folder into the repository root (or subfolder).
3. Enable GitHub Pages from the repository settings.
4. Share the resulting Pages URL with the couple.

Do not add counsellor notes or profile files to the published page.
