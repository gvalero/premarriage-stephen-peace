# Stephen & Peace Spiritual Life Together Homework Page

This folder is a GitHub Pages-ready static site for Session 4 homework.

Three jobs the homework does:
1. Closes the loop on the **repair covenant** we didn't get to in Session 3.
2. Looks at **how the wedding/honeymoon decision was made** as a live case study.
3. Sets up the next chapter the couple chose: **Your Spiritual Life** (book pp. 60-67).

The chapter PDF (`reading-spiritual-life-pp-60-67.pdf`) ships alongside the page so
they can read without needing the physical book.

Important privacy note: this page does not send responses anywhere by itself. GitHub Pages is static and has no private form backend. The form stores answers in the browser only, then lets Stephen and Peace copy or download their responses and send them back manually.
